使用BlueXP backup and recovery，將 Microsoft SQL Server 應用程式資料複製到相同或不同的虛擬機，用於開發、測試或保護目的。您可以從 Microsoft SQL Server 工作負載的即時快照或現有快照建立複製。

* 重新整理實體複本
* 分割實體複本
* 刪除克隆
