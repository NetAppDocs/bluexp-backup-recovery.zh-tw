= BlueXP backup and recoveryVMware 工作負載的已知限制
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
已知限制指出本產品版本不支援的平台、裝置或功能、或是無法與產品正確互通的平台、裝置或功能。請仔細檢閱這些限制。

BlueXP backup and recovery中的 VMware 工作負載預覽版不支援以下操作：

* 山
* 解除安裝
* 恢復至備用位置
* 恢復 VMDK
* 附加 VMDK
* 分離 VMDK
* vVol 支持
* NVMe 支持
* 電子郵件集成
* 編輯政策
* 編輯保護群組
* 基於角色的存取控制 (RBAC) 支持
